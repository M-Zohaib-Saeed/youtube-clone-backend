# 🎥 Video Streaming Backend (YouTube-like API)

This is a backend API for a video streaming platform similar to YouTube. It is built using **Node.js, Express, and MongoDB** and supports features like user authentication, video upload, and streaming.

---

## 🚀 Features

- User registration and login (JWT Authentication)
- Upload videos
- Stream videos
- Like / dislike videos
- Add comments
- User profile management
- Secure password hashing
- RESTful API structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Token (JWT)
- bcrypt.js
- Multer (for file uploads)
